Sobre

Olá, Eu sou a Maytê, moro em Praia Grande / SP sou técnica de informática e graduada em automação industrial, ambos pelo Instituo Federal e atualmente curso análise e desenvolvimento de sistemas pela Fatec. Possuo mais de 10 anos de experiência em áreas administrativas e atendimento ao cliente, onde adquiri habilidades valiosas em comunicação e resolução de problemas. Sou uma desenvolvedora de software em desenvolvimento, estou em uma emocionante jornada de transição de carreira para a área de Tech.

Minhas formações me proporcionaram uma base sólida em lógica de programação, orientação a objetos, análise de requisitos, banco de dados e desenvolvimento de software. Tendo contato com: Python, C#, Java, PHP, JavaScript, HTML, CSS, GIT, Flutter, Laravel, SQL, C++, Delphi. Alguns projetos da graduação e de outros cursos estão em meu Github.

Estou constantemente aprimorando minhas habilidades por meio de cursos, bootcamps, mentorias e programas de desenvolvimento profissional que contribuam com a minha evolução profissional. Me sinto animada para aplicar minhas habilidades nessa nova carreira e contribuir para projetos de tecnologia da informação

O Início

O meu interresse pela área de informática, aliado ao desejo de aprimorar minhas habilidades como desenvolvedora de software e amor pelo aprendizado, me levaram a participar do programa [Elas na_tech].

Projetos

Durante a jornada do programa, fomos constantemente desafiadas a aplicar nossos conhecimentos, resultando na criação dos projetos a seguir:

Cachorros

Projeto concebido utilizando HTML e CSS, com um foco específico na criação de uma galeria de imagens, aproveitando o carrossel fornecido pelo Bootstrap e inserção de favicon. Além disso, foi implementado um menu hamburguer para garantir uma navegação suave em telas menores, priorizando a responsividade. A manipulação do tamanho das imagens, bem como a customização do menu, do logotipo e dos botões, foram planejadas para aprimorar a estética e a interatividade do site.

Projeto Código
Groot

Projeto criado utilizando HTML e CSS com o objetivo de implementar um menu hamburguer responsivo, que apenas é visualizado em telas menores, além de criar um design com header e footer atrativos, e adicionar efeitos hover para melhorar a interatividade. O projeto concentra-se em garantir que o site seja acessível e agradável em uma variedade de dispositivos, enquanto oferece uma experiência de usuário atraente.

Projeto Código
Rainhas Africanas

Projeto elaborado com a utilização das linguagens HTML e CSS com o intuito de aplicar uma variedade de técnicas, incluindo a combinação de cores, efeitos hover, ajuste de tamanhos de imagens, inserção de imagens de fundo, a inclusão de links externos e a implementação de tags semânticas. Ele representa uma oportunidade para explorar e aprimorar habilidades em design e desenvolvimento web, oferecendo uma experiência prática e envolvente..

Projeto Código
Pantera Negra

Projeto desenvolvido utilizando HTML e CSS com o propósito de aplicar uma ampla variedade de técnicas e conceitos. Alguns dos aspectos abordados incluem a utilização do Normalize para limpeza de pré-configurações, a inserção de logotipos e menus, a configuração de fontes para aprimorar a estética, a inclusão de vídeos para tornar o conteúdo mais envolvente, a criação de botões interativos, a aplicação de layouts flexíveis com Flexbox, além da implementação da responsividade para garantir uma experiência consistente em diferentes dispositivos. Este projeto representa uma oportunidade abrangente para explorar e aprimorar habilidades de desenvolvimento web, abrangendo diversos aspectos do design e da funcionalidade.

Projeto Código
Responsividade

Projeto foi criado com HTML e CSS, tendo como principal objetivo a aplicação dos conceitos de responsividade com abordagem 'mobile first'. Isso significa que o site foi projetado para ser completamente funcional e visualmente agradável em uma variedade de tamanhos de tela, permitindo que os usuários desfrutem da experiência em dispositivos diversos. Este enfoque na responsividade garante que o site seja acessível e eficaz, independentemente do dispositivo utilizado.

Projeto Código
Frutas Vermelhas

Projeto criado com HTML e CSS, com o objetivo de aplicar diversos conceitos essenciais de design e layout. Durante a sua elaboração, exploramos as funcionalidades do layout flexbox, aplicamos efeitos de filtro grayscale e transformações de escala (scale), configuramos o logotipo e estilizamos o rodapé. O resultado final é um exemplo prático e visualmente atraente que demonstra a aplicação desses conceitos-chave no design de páginas web..

Projeto Código
Calculadora JS

Projeto desenvolvido com a utilização das linguagens HTML, CSS e JavaScript, com o propósito de aplicar os princípios fundamentais do JavaScript por meio do desenvolvimento de uma calculadora simples. Nele, exploramos os conceitos do DOM (Modelo de Objeto de Documento), oferecendo uma experiência prática e interativa.

Projeto Código
Javascript

Projeto foi desenvolvido com HTML, CSS e JavaScript com o propósito de aplicar conceitos essenciais das linguagens e demonstrar o uso de estruturas de controle condicionais em JavaScript. Neste projeto, o usuário é convidado a inserir um número, e em seguida, o programa verifica se o número atende às condições de estar no intervalo de 0 a 100 e ser divisível por 2, ou seja, um número par.
